# Employee Attendance System

A web-based employee attendance system developed using Laravel.  
The system allows employees to check in and check out using GPS location and selfie verification, while administrators can monitor attendance data through a dashboard.

## Tech Stack
- Laravel
- PHP
- MySQL
- Blade Template
- Bootstrap
- JavaScript

## Features
- User authentication login and register
- Role-based access admin and employee
- GPS-based attendance check-in and check-out
- Selfie photo capture
- Attendance history
- Admin dashboard

## Screenshots
- Login Page
- Dashboard

## Installation
- Clone this repository
- Run composer install
- Copy .env.example to .env
- Configure database
- Run php artisan migrate
- Run php artisan serve
